# Danh sách kiểu dữ liệu (data type)

_Product, khách hàng (client), tồn kho (inventory), kho (warehouse), giá theo bậc (tier pricing), đơn hàng (order), lịch sử đơn (order history)_

**EN / gộp tài liệu:** Đây là bản **canonical**. Cột tiêu đề có thể dùng song ngữ (Area / Tables / Field types); nội dung bảng trùng với **bản tiếng Anh** trước khi gộp — bản EN tách file đã **gỡ khỏi repo (2026-05)** để tránh hai bản drift.
