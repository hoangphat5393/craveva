## 1. Getting Started

### 1.1 Logging In

- Use the login page provided by your organization.
- If you forgot your password, use the “Forgot Password” option.

### 1.2 First-Time Setup Checklist

- Update your profile details (name, email, phone).
- Set your timezone and preferred language.
- Configure notifications (email/app) if your organization uses them.
- Review your role access (some menus may be hidden based on permissions).

---

## 2. Navigation Basics

### 2.1 Main Areas

- Dashboard: Your starting point for daily work and key summaries.
- Sidebar Menu: Access modules such as HR, Work, CRM, Finance, Support.
- Global Search: Quickly find items (where enabled) like clients, projects, invoices.

### 2.2 Common Screen Patterns

- List Page: Shows records with search, filters, sorting, and pagination.
- Create/Edit Form: Used to add or update information.
- Details Page: Shows a record with related tabs (activity, files, notes, history).
- Actions Menu: Quick actions like edit, archive, delete (permission-based).

---

## 3. Users, Roles, and Permissions

### 3.1 Roles

Roles define what a user can see and do. Common examples:

- Admin: Full access to most company features.
- Manager: Department-level access (people, work items, reporting).
- Employee/Staff: Daily operations (tasks, attendance, requests).
- Client: Limited access to their own projects, invoices, tickets (if enabled).

### 3.2 Permissions

- Permissions control specific actions (view, create, edit, delete, export).
- If you cannot see a menu or a button, it usually means you do not have access.

## 4. Core Workflows (Most Common Tasks)

### 4.1 Add a Client (Sales/CRM)

- Open the Clients module.
- Select Add New.
- Fill in client details (company name, contact person, email, phone).
- Save, then add notes and assign an owner if your team uses ownership.

### 4.2 Create a Project (Work)

- Open the Projects module.
- Select Add New.
- Set project name, start date, end date (optional), and assign members.
- Save, then add tasks and milestones.

### 4.3 Manage Tasks (Work)

- Create tasks under a project or in the Tasks module.
- Assign owners, due dates, priority, and status.
- Use comments and file attachments for collaboration.

### 4.4 Attendance and Leave (HR)

- Attendance: Use your dashboard tools to check in/out if enabled.
- Leave: Submit leave requests, track approval status, and view balances.

### 4.5 Create an Invoice and Record Payment (Finance)

- Open the Invoices module.
- Create invoice for a client and add line items.
- Send invoice (email option may be available).
- Record payment when received (based on your organization’s process).

### 4.6 Raise and Track Tickets (Support)

- Open the Tickets module.
- Create a ticket with subject, description, and priority.
- Attach screenshots or files if needed.
- Track progress through ticket status updates.

## 6. Company Settings (Admin)

This section is for company Admins who can configure how the system behaves for everyone. Some settings appear only when a module is enabled.

### 6.1 Company Profile

- Update company name, email, phone, and website.

### 6.2 Business Addresses

- Add one or more business addresses.
- Choose a default address used in documents and templates.

### 6.3 App Settings

- Set company timezone, date format, and time format.
- Set default language for the company.
- Set default currency.
- Adjust list/page limits (how many rows display per page).
- Optional: enable dashboard clock.
- Optional: allow employees to export data.
- Optional (if available): client signup settings (allow signup, admin approval).
- Optional (if available): file upload limits (max file size, allowed types, max files).

### 6.4 Profile Settings

- Update your personal profile details and preferences.

### 6.5 Notifications

- Email (SMTP): configure outgoing email and send a test email.
- Email notifications: choose which events send emails.
- Slack notifications: connect Slack and choose events.
- Push notifications: connect push provider and send a test notification.
- Pusher/Realtime (if used): configure realtime events.

### 6.6 Currency Settings

- Manage currencies and currency formatting.
- Configure exchange-rate provider key and refresh exchange rates.

### 6.7 Payment Settings

- Configure online payment gateway credentials (based on enabled gateways).
- Configure offline payment methods (bank transfer, cash, etc.).

### 6.8 Finance Settings (Invoices/Estimates/Orders)

- Configure invoice numbering (prefix, format).
- Manage invoice templates and what client details show on documents.
- Configure default notes/terms and other document info.

### 6.9 Tax Settings

- Create and manage tax rates used in invoices and orders.

### 6.10 Ticket Settings (Support)

- Manage ticket agents and agent groups.
- Configure ticket groups, ticket types, and channels.
- Manage reply templates.
- Configure email sync (if enabled) and ticket assignment rules (including round-robin).

### 6.11 Project Settings

- Configure project categories.
- Configure project statuses and default behaviors.

### 6.12 Attendance Settings (HR)

- Configure attendance rules (working hours, breaks, late marks).
- Configure shifts and shift rotations (if enabled).
- Optional: location/geofencing rules (if enabled).

### 6.13 Leave Settings (HR)

- Configure leave policies, approvals, and visibility.
- Manage leave types (annual, sick, etc.).

### 6.14 Custom Fields

- Add custom fields to forms (clients, projects, tickets, employees, etc.).

### 6.15 Roles & Permissions

- Create roles and control what each role can view/do.
- Decide what clients, employees, and managers can access.

## 7. Best Practices

- Keep records complete: add notes, attachments, and clear descriptions.
- Use consistent naming: projects, invoices, and tasks are easier to search.
- Assign ownership: ensure each task/ticket has a responsible person.
- Maintain statuses: update task and ticket statuses to reflect reality.
- Use filters and saved views (if available) to speed up daily work.

## 8. Troubleshooting

### 8.1 I Can’t See a Module or Button

- It is usually permission-based.
- Contact your Admin to confirm your access.

### 8.2 Data Is Missing or Not Updated

- Refresh the page and re-check filters.
- Confirm date ranges and status filters.
- If it persists, report it to your Admin.

### 8.3 Notifications Not Working

- Check your profile notification settings.
- Check spam/junk folder for emails.
- Confirm your email address is correct.
