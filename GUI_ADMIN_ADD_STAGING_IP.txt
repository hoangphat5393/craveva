   IP cần thêm:  35.240.234.226
   Tên (Name):   staging-vm   (tùy chọn)

Cách làm nhanh:
1. Vào https://console.cloud.google.com/sql/instances
2. Chọn project chứa database (instance có IP 136.110.52.19)
3. Bấm vào TÊN instance đó
4. Tab "Connections" → phần "Authorized networks" → "ADD NETWORK"
5. Điền Network: 35.240.234.226    Name: staging-vm
6. Bấm Done → Save
