# Line Deployment Package

This package contains everything you need to deploy your Craveva AI agent to line.

## Contents

- `webhook-config.json` - Webhook configuration
- `verification-code.js` - Webhook verification code
- `setup-guide.md` - Step-by-step setup instructions
- `api-credentials.md` - Where to find your API credentials

## Quick Start

1. Configure webhook URL: `https://ai.craveva.com/api/v1/webhooks/line`
2. Use verification code: `VERIFY_CODE`
3. Follow the setup guide in `setup-guide.md`

## Support

For detailed instructions, see `setup-guide.md`.
