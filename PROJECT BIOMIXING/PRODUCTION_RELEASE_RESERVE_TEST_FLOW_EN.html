Production Test Run Flow (Release Reserve)

This flow is for PM and testers to run UAT after enabling raw-material reserve at Release.

  • Main scope: PR-01 to PR-05 (automated) and PR-10, PR-11, PR-12 (manual).
  • DB checks: stock_reservations and warehouse_product_batches.reserved_quantity.
