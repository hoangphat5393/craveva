# Custom System Deployment Package

This package contains everything you need to integrate your Craveva AI agent into your custom system.

## Integration Methods Included

- **Widget**
- **REST API**
- **Webhook**

## Quick Start

1. **🧪 Test First!** If widget is included, double-click `test.html` to test immediately
2. Choose your integration method
3. Follow the README in the corresponding folder
4. Test your integration

## Folder Structure

- `widget/` - Widget embedding code in all formats
- `api/` - REST API documentation and examples
- `webhook/` - Webhook setup and configuration
- `test.html` - **✨ Double-click to test immediately!** Standalone test file with embedded widget


## Support

Each integration method has its own README with detailed instructions.
