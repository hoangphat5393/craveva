# Multi-Language Integration Code

This directory contains ready-to-use integration code for your Craveva AI agent in multiple programming languages and frameworks.

## Available Languages

- **Python** - Python 3.x with requests library
- **Laravel/PHP** - Laravel service class
- **Node.js** - Node.js/Express with axios
- **Java** - Java/Spring Boot service
- **C# / .NET** - C# service for .NET
- **Ruby** - Ruby class with HTTParty

## Quick Start

1. Navigate to your preferred language folder
2. Copy the code file
3. Follow the README in that folder for installation instructions
4. Update the API key and base URL in the code
5. Start using the agent in your application!

## API Key

Your API key is included in the `api/` folder (if API integration is selected).

## Support

Each language folder contains:
- The integration code file
- A README with specific installation instructions
- Usage examples

Choose the language that matches your tech stack!
