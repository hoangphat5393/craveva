# React Installation Guide

## Installation

1. Install React: `npm install react react-dom`
2. Copy the component code from `CravevaAgent.tsx`
3. Import and use in your React app

```tsx
import { CravevaAgent } from './CravevaAgent';

function App() {
  return (
    <div>
      <CravevaAgent agentId="YOUR_AGENT_ID" />
    </div>
  );
}
```
