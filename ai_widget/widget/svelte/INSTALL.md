# Svelte Installation Guide

## Installation

1. Install Svelte: `npm install svelte`
2. Copy the component code from `CravevaAgent.svelte`
3. Import and use in your Svelte app

```svelte
<script>
  import CravevaAgent from './CravevaAgent.svelte';
</script>

<CravevaAgent agentId="YOUR_AGENT_ID" />
```
