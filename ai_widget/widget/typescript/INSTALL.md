# Typescript Installation Guide

## Installation

1. Install TypeScript: `npm install -g typescript`
2. Copy the code from `widget.ts`
3. Import and use in your TypeScript project

```typescript
import { deployCravevaAgent } from './widget';

deployCravevaAgent({
  agentId: 'YOUR_AGENT_ID',
  apiBase: 'YOUR_API_BASE'
});
```
