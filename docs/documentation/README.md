# Documentation Workflow

Tài liệu trong thư mục này định nghĩa cách tổ chức và duy trì Markdown của dự án.

- [`MD_HEADER_TEMPLATE.md`](MD_HEADER_TEMPLATE.md) - metadata và section chuẩn cho tài liệu chức năng mới.
- [`MD_WORKFLOW_QUICKSTART.md`](MD_WORKFLOW_QUICKSTART.md) - lệnh kiểm tra index, master guide và AI context.

Các tài liệu nghiệp vụ vẫn đặt trong `FUNC_LOGIC`, `FUNC_IMPROVE`, `FUNC_BUG`,
`SPECIFICATION` hoặc module tương ứng.
