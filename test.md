Nếu bạn dùng... Nên chọn bản... Lý do
IIS (Windows) NTS IIS chạy PHP thông qua FastCGI, môi trường này ưu tiên NTS để đạt hiệu suất cao nhất.
Nginx (Windows) NTS Nginx cũng dùng FastCGI (php-cgi.exe), nên NTS là bắt buộc.
Apache (Windows) TS Nếu bạn cấu hình Apache sử dụng php8ts.dll.
Chạy lệnh Terminal NTS Các lệnh như php artisan, composer chạy tốt và nhanh hơn với NTS.
